Title-
Maharashtra Vegetable Market Analysis

Short Description-
This project analyzes vegetable prices and production data across Maharashtra to identify high-profit and low-risk crops for farmers. It combines mandi price trends with production insights to support better agricultural decision-making.

---
Tech Stack-
- Python (Pandas, Selenium, BeautifulSoup) – Data scraping & cleaning  
- SQL (MySQL) – Data querying & transformation  
- Power BI – Dashboard & data visualization  
- Excel – Initial data handling  

---
Data Sources-
- Mandi price data (Agmarknet / KisanDeals scraping)  
- District-wise production data (agriculture datasets)  

---
Features-
- 📈 Price trend analysis for vegetables (Jan–Apr 2026)  
- 🗺️ District-wise price comparison  
- 📊 Production analysis by crop and category  
- ⚖️ Profit score calculation (price × production)  
- ⚠️ Supply gap detection (high price + low arrivals)  
- 📌 Interactive Power BI dashboard with filters and KPIs  

---
Key Questions Answered-
- Which vegetables have the highest average price?  
- Which districts generate the most profit?  
- Which crops are stable vs highly volatile?  
- Where is the supply-demand gap highest?  
- Which vegetables are most profitable to grow?  

---
Business Impact & Insights-
- Identified high-profit crops like onion and green chilli based on price trends  
- Highlighted districts with strong earning potential for farmers  
- Detected supply gaps indicating opportunities for increased production  
- Provided data-driven crop recommendations to reduce risk and improve profitability  

---
Dashboard Highlights-
- KPI cards (Highest price crop, best profit district, highest production, supply gap)  
- Price trend line charts  
- Commodity-level comparison table with conditional formatting  
- District-wise analysis visuals
